函数参数传递：
    基本数据类型：值传递
    数组：引用传递
    对象类型：引用传递
    String对象: 值传递，因为其内部的字符保存数组为final

数组：
    int[] Array=new int[]{1,2,3}       //初始化
    int len=Array.length                //返回数组长度
字符串：
    int index=str1.indexOf(str2);    //在str1中查找str2并返回下标，没有找到返回-1，str2是空串返回0
    String subs=str1.substring(start,end);       //返回下标为start到end之间的字串（包括start，不包括end）
    int len=subs.length();      //返回字符串长度
    subs.equals(str1)；       //比较两个字符串是否一样，'=='和'equals'都是比较对象的地址，只是equals可以重写
